## 📖 Sobre

Este repositório reúne os códigos, exercícios e projetos desenvolvidos durante o **Curso de JavaScript** do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado pelo professor **Gustavo Guanabara**.

O curso aborda JavaScript do zero, com foco em lógica de programação, manipulação do DOM, eventos, funções e muito mais — tudo na prática.

---

## 📂 Estrutura do Repositório

```
javascript-cursoemvideo/
├── aula04/          # Primeiros passos com JS no navegador
├── aula06/          # Variáveis, tipos e operadores
├── aula09/          # Condicionais (if/else, switch)
├── aula10/          # Repetições (while, for)
├── aula11/          # Funções
├── aula12/          # Arrays e objetos
├── ex012/           # Exercício da aula 12

└── README.md
```

## 📚 Conteúdo das Aulas

| Pasta | Conteúdo |
|---|---|
| `aula04` | Primeiros scripts — `alert()`, `prompt()`, `console.log()` |
| `aula06` | Variáveis (`var`, `let`, `const`), tipos primitivos e operadores |
| `aula09` | Estruturas condicionais — `if`, `else`, `else if`, `switch` |
| `aula10` | Laços de repetição — `while`, `do while`, `for` |
| `aula11` | Funções — declaração, parâmetros, retorno e escopo |
| `aula12` | Arrays, métodos de array e introdução a objetos |
| `ex012` | 💪 Exercício prático — fixação do conteúdo da aula 12 |
| `Nacionalidade` | 🗺️ Projeto: identificação de nacionalidade por entrada do usuário |

## 🚀 Como Visualizar os Projetos

Não precisa instalar nada! Todos os exercícios rodam direto no navegador.

1. Clone o repositório:
```bash
git clone https://github.com/eduardoferreirraa/javascript-cursoemvideo.git
```

2. Abra a pasta da aula desejada e clique duas vezes no arquivo `index.html`

Ou acesse diretamente pelo GitHub Pages (se disponível).
